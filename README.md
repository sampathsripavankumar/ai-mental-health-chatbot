#  AI Mental Health Chatbot

> Final Year B.Tech Project | Artificial Intelligence & Machine Learning

##  Overview
An AI-powered chatbot that detects the user's **emotion** from text using 
NLP and responds with empathy. Built using HuggingFace Transformers and Gradio.

##  Problem Statement
Mental health support is inaccessible to millions. This chatbot provides 
instant, empathetic responses based on real-time emotion detection.

##  Features
-  Detects 7 emotions: Joy, Sadness, Anger, Fear, Disgust, Surprise, Neutral
-  Empathetic chatbot responses for each emotion
-  Interactive web UI using Gradio
-  Powered by DistilRoBERTa (HuggingFace)
